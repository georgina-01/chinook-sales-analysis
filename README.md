# Chinook Music Store - Sales Analysis
Exploratiry analysis of a digital music store using SQL and Python, uncovering revenue trends, top-performing products, and employee performance.

## Tools
Python
SQLite
Pandas
Matplotlib
Jupyter Notebook

## Key Questions
- Which tracks sell the most?
- Which countries generate the most revenue?
- Who are the top-performing sales employees?
- How has revenue trended month over month?
- Which customers are high-value vs low-value?

## Key Findings
- The USA drives ~22% of total revenue, the single largest market
- The top 5 countries account for over 50% of all revenue
- Track popularity is evenly distributed, no single breakout hit
- Revenue per customer varies by sales rep, suggesting a training opportunity
- Tracks purchased strongly correlates with total spend (r ≈ 0.99)

## How to Run
1. Clone the repo
2. Install dependencies
  pip install pandas matplotlib jupyter
3. Open the notebook
   'notebook/Chiook Music Store - Sales Analysis'

## Dataset
[Chinook Database](https://github.com/lerocha/chinook-database) —
a sample database representing a digital media store with 11 tables
covering artists, albums, tracks, invoices, customers, and employees.
